== Description ==

Use shortcode [neswletter_form] to display the newsletter form popup.

Cookie for the current user email will be stored for 7 days.

== Installation ==

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `newslettr.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

== Use ==

Use shortcode [neswletter_form] to display the newsletter form anywhere.
Put the shortcode into header or footer area to access popup all over the site.
